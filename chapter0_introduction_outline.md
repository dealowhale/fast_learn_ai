  - 内存: 8GB RAM (推荐16GB)

## 0.4 必备的Python基础知识回顾 (0.5页)

### 0.4.1 Python环境检查 (0.25页)
**学习目标**: 确保Python环境正确配置

**内容要点**:
- **Python版本确认**:
  ```bash
  python --version  # 应该是3.8+
  pip --version     # 确认包管理器可用
  ```
- **虚拟环境创建**:
  ```bash
  python -m venv ai_env
  # Windows
  ai_env\Scripts\activate
  # macOS/Linux
  source ai_env/bin/activate
  ```
- **基础库安装**:
  ```bash
  pip install numpy pandas matplotlib jupyter
  ```
